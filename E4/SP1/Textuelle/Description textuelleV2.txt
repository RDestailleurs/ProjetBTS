Description textuelle


1- Le client arrive sur la page d'accueil 
2- Le client clique sur le lien tarifs dans le menu 
3- Le système affiche la page tarifs
4- Le client clique sur le lien horaires dans le menu
5- Le systeme affiche la page horaires
6- Le Client selectionne une liaison ainsi qu'une date 
7- Le système affiche les traversées disponibles pour cette liaison à cette date
8- le client coche la traversée qu'il désire puis clique sur le bouton "réserver"
9- Le systeme regarde si le client est connecté 
10- Le client n'est pas connecté
11- Le système redirige le client sur la page compte
12- Le client choisit de se connecter 
13- Le systeme redirige le client sur la page de reservation
14- Le client renseigne les informations demandés et clique sur le bouton "enregistrer la réservation"
15- Le système vérifie que les entrées texte ne sont pas vides
16- Le système affiche un bilan de la réservation 
17- Le client procède au paiement



Alternatif :


7b- Aucune traversée n'est disponible, le systeme affiche un message d'erreur
10b- le client est connecté 
10b- le systeme redirige le client sur la page reservation 
12b- Le client choisit de s'inscrire 
12b- Le systeme redirige le client sur la page d'inscription
15b- Certaines entrées textes sont vides, le système demande au client de renseigner les champs demandés
